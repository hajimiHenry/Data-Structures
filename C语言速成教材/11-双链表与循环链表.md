# 11 双链表与循环链表

单链表的两种常见变体。理解了 10 章，这章是「改个节点结构」的练习。

## 11.1 双链表：多一个前驱指针

单链表的痛点：只能往后走，要往回走得从头找。双链表每个节点多一个 `prev` 指向**前一个**节点，双向都能走。

```c
typedef struct DNode {
    int data;
    struct DNode *prev;   // 前驱
    struct DNode *next;   // 后继
} DNode;
```

```
NULL ← [prev|1|next] ⇄ [prev|2|next] ⇄ [prev|3|next] → NULL
       头                                  尾
```

- 头节点的 `prev == NULL`，尾节点的 `next == NULL`。
- 删除某节点 O(1)（单链表删除还要找前驱 O(n)，双链表直接用 `p->prev`）——这是双链表的主要优势，代价是每节点多一个指针。

### 双链表插入（在 p 之后插新节点）

```c
void insert_after(DNode *p, int data) {
    DNode *n = malloc(sizeof(DNode));
    n->data = data;
    n->prev = p;
    n->next = p->next;          // 接后面
    if (p->next) p->next->prev = n;   // 后面的 prev 回指（小心尾部 NULL）
    p->next = n;               // 前面接上
}
```

### 双链表删除（删节点 p）

```c
DNode *remove_node(DNode *head, DNode *p) {
    if (p->prev) p->prev->next = p->next;    // 前驱的后继跳过 p
    else        head = p->next;              // p 是头，换头
    if (p->next) p->next->prev = p->prev;    // 后继的前驱跳过 p
    free(p);
    return head;
}
```
不用遍历找前驱，直接 `p->prev`——这是双链表 O(1) 删除的来源。

## 11.2 循环链表：尾指向头

把单链表尾节点的 `next` 指向头，就成了循环链表。

```
head → [1] → [2] → [3] ┐
        ↑───────────────┘
```

```c
// 建循环链表（尾插后把尾连回头）
Node *last = ...;          // 找到尾
last->next = head;         // 尾指向头，成环
```

**遍历循环链表要注意**：不能用 `while (cur != NULL)`（永远不到 NULL 死循环），要用 `do-while` 或标记起点：

```c
void print_circular(Node *head) {
    if (!head) return;
    Node *cur = head;
    do {
        printf("%d ", cur->data);
        cur = cur->next;
    } while (cur != head);   // 回到起点就停
}
```

> 应用：约瑟夫环问题（报数出列）就是循环链表的经典场景。

## 11.3 双向循环链表

两者结合：头尾双向相连。Linux 内核链表、LRU 缓存常用这种。原理同上，节点维护 prev/next，尾的 next 指向头、头的 prev 指向尾。

## 易错点

1. 双链表插入/删除要**双向都改**（prev 和 next 各两处），漏一处就断链。
2. 删头/删尾边界：判 NULL 前先看 `p->prev` / `p->next` 是否存在。
3. 循环链表遍历别用 `!= NULL`，会死循环。
4. 双链表节点 free 前，先断开它和邻居的联系。

## 例题

**例 11.1** 双链表删节点，为什么不用像单链表那样遍历找前驱？
答：因为 `p->prev` 直接就是前驱，O(1)。单链表没有 prev 指针，只能从头遍历。

**例 11.2** 单链表 `1→2→3` 改成循环链表后，从头遍历会打印什么（用 `do{...}while(cur!=head)`）？
答：`1 2 3`，打印一圈回到 head 停。

## 小练习

1. 写双链表的 `push_front`（头插，返回新头）。
2. 写循环单链表的长度计算（别死循环）。

<details><summary>答案</summary>

```c
// 1
DNode *d_push_front(DNode *head, int data) {
    DNode *n = malloc(sizeof(DNode));
    n->data = data;
    n->prev = NULL;
    n->next = head;
    if (head) head->prev = n;
    return n;
}

// 2
int circular_length(Node *head) {
    if (!head) return 0;
    int n = 0;
    Node *cur = head;
    do { n++; cur = cur->next; } while (cur != head);
    return n;
}
```
</details>

双链表/循环链表是单链表的自然扩展。核心还是 10 章那套「找前驱、改指针、释放节点」的心法。
