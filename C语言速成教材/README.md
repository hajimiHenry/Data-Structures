# C 语言速成教材（408 数据结构向）

为你量身：你跟过翁恺老师浙大 C 课，但隔了一学期**基础也忘了不少**，而且**链表、动态内存、递归从没学过**。现在要准备 408，需要用 C 讲数据结构。

**所以这本教材两部分**：
- **基础篇（01–07）：快讲，但讲透**。你不记得了，所以每个概念都从「这是什么」讲起，但节奏快、例子直接、不灌水。能看懂就快速往下走。
- **重点篇（08–12）：深讲**。动态内存 → 自引用结构体 → 单链表 → 递归，这是你完全没学过、且数据结构最依赖的部分，逐行讲，每章配可运行完整代码。
- **13 函数指针与 qsort**：补充，排序比较常用。

## 怎么用

每章固定结构：**正文（概念+示例）→ 易错点 → 例题（带解析）→ 小练习（答案在本章末）**。

- 基础篇：先自己读，能直接做例题就跳过该节。
- 重点篇：代码一定要**自己敲一遍、跑一遍**，最好开 `-fsanitize=address` 查内存错误。
- 10 单链表是检验 08/09 是否真懂的试金石；12 递归是看懂二叉树的前提。

## 环境（Mac 终端）

```bash
# 编译运行
clang a.c -o a && ./a
# 开警告 + 内存检查（学动态内存时强烈推荐）
clang -Wall -fsanitize=address a.c -o a && ./a
```
