# 10 单链表完整实现 ⭐⭐

这章把前面学的**结构体 + 指针 + 动态内存 + 自引用**全用上，实现一条完整的单链表。**如果这章你能自己敲出来跑通，前面的知识就过关了。**

## 10.1 节点与头指针

```c
#include <stdio.h>
#include <stdlib.h>

typedef struct Node {
    int data;
    struct Node *next;
} Node;
```

链表的唯一入口是**头指针 `head`**：
- `head == NULL` 表示空链表。
- `head` 指向第一个节点，第一个节点的 `next` 指向第二个，…… 最后一个节点的 `next == NULL`。

## 10.2 建一个新节点

所有操作都从造节点开始：

```c
Node *make_node(int data) {
    Node *n = malloc(sizeof(Node));
    if (n == NULL) { fprintf(stderr, "malloc failed\n"); exit(1); }
    n->data = data;
    n->next = NULL;
    return n;
}
```

## 10.3 头插法建表

新节点插到最前面，`head` 始终指向最新节点。

```c
// 方式一：返回新头指针（推荐，最简单）
Node *push_front(Node *head, int data) {
    Node *n = make_node(data);
    n->next = head;     // 新节点指向原来的头
    return n;           // 新节点成为新头
}

// 用法
Node *head = NULL;
head = push_front(head, 3);   // 链表：3
head = push_front(head, 2);   // 链表：2 3
head = push_front(head, 1);   // 链表：1 2 3
```

> 头插法建出来的链表，顺序和插入顺序**相反**（后插的在前）。这点常考。

**为什么返回新 head？** 复习 05/03 章：函数改不了调用方的 `head` 本身（值传递）。`push_front` 要把 head 改成指向新节点，所以**返回新值让调用方重新赋值**。这是最直观的写法。

**方式二：二级指针**（不用返回值，函数直接改调用方的 head）：

```c
void push_front2(Node **head, int data) {
    Node *n = make_node(data);
    n->next = *head;    // *head 就是调用方的头指针
    *head = n;          // 改调用方的 head！
}

// 用法
Node *head = NULL;
push_front2(&head, 3);   // 传 head 的地址
```

> 两种写法二选一即可。**推荐先用「返回新头」**，简单不易错；看数据结构教材遇到二级指针写法时也认得出来。

## 10.4 尾插法建表

新节点插到末尾。需要先找到当前最后一个节点（next 为 NULL 的那个）。

```c
Node *push_back(Node *head, int data) {
    Node *n = make_node(data);
    if (head == NULL) return n;   // 空表，n 就是新头

    Node *cur = head;
    while (cur->next != NULL)     // 走到最后一个（不是 NULL！）
        cur = cur->next;
    cur->next = n;                // 接上
    return head;                  // 头没变
}

// 用法
Node *head = NULL;
head = push_back(head, 1);   // 1
head = push_back(head, 2);   // 1 2
head = push_back(head, 3);   // 1 2 3
```

**关键区别**：
- 遍历找最后一个：`while (cur->next != NULL)`（停在最后一个节点上，不是 NULL）。对比 10.x 例题 9.4。
- 空表单独处理：空表时没有「最后一个节点」，新节点直接成为头。

> 尾插 O(n)（每次都要走到底）。优化用「尾指针 tail」记最后位置可 O(1)，这里先懂原理。

## 10.5 遍历与长度

```c
void print_list(Node *head) {
    for (Node *cur = head; cur != NULL; cur = cur->next)
        printf("%d ", cur->data);
    printf("\n");
}

int length(Node *head) {
    int n = 0;
    for (Node *cur = head; cur; cur = cur->next) n++;
    return n;
}
```

## 10.6 查找

```c
Node *find(Node *head, int target) {
    for (Node *cur = head; cur; cur = cur->next)
        if (cur->data == target) return cur;
    return NULL;
}
```

## 10.7 在第 i 个位置插入

插入到下标 i 处（0 表示头插）。**关键：要改前一个节点的 next**。

```c
Node *insert_at(Node *head, int i, int data) {
    if (i == 0) return push_front(head, data);   // 头插

    Node *cur = head;
    for (int k = 0; k < i - 1; k++) {             // 走到第 i-1 个
        if (cur == NULL) return head;             // i 越界，不变
        cur = cur->next;
    }
    if (cur == NULL) return head;                  // i 越界

    Node *n = make_node(data);
    n->next = cur->next;    // 新节点接上后面的
    cur->next = n;          // 前一个接上新节点
    return head;
}
```

**链表插入/删除的核心心法**：你要操作的是**前一个节点**的 `next` 指针。所以先走到「前一个节点」(i-1)，再改它的 next。这和数组完全不同（数组是直接按下标），要专门建立直觉。

**两步接指针的顺序**：先 `n->next = cur->next`，再 `cur->next = n`。顺序反了（先 `cur->next = n`）会丢掉后面的链。画图就懂：

```
插入 n 到 cur 之后，原 cur->next 是 b：
正确顺序：
  n->next = b;     n: [data|b]
  cur->next = n;   cur: [..|n]  →  cur → n → b
```

## 10.8 删除节点

删除第一个值为 val 的节点。同样要找**前一个节点**。

```c
Node *remove_val(Node *head, int val) {
    if (head == NULL) return NULL;

    // 特殊：删头节点
    if (head->data == val) {
        Node *tmp = head;
        head = head->next;
        free(tmp);
        return head;
    }

    // 找前一个节点
    Node *cur = head;
    while (cur->next != NULL && cur->next->data != val)
        cur = cur->next;

    if (cur->next == NULL) return head;   // 没找到

    Node *tmp = cur->next;        // 要删的节点
    cur->next = tmp->next;        // 前一个跳过它
    free(tmp);                    // 释放
    return head;
}
```

**删除三步**：① 找前驱；② 让前驱的 next 跳过待删节点（`cur->next = tmp->next`）；③ free 待删节点。删头单独处理（因为头没有前驱，要改 head）。

## 10.9 释放整表

```c
void free_list(Node *head) {
    Node *cur = head;
    while (cur) {
        Node *nx = cur->next;   // 先存
        free(cur);
        cur = nx;
    }
}
```

## 10.10 完整可运行代码

把上面拼起来，这是你能直接编译运行的完整单链表：

```c
#include <stdio.h>
#include <stdlib.h>

typedef struct Node {
    int data;
    struct Node *next;
} Node;

Node *make_node(int data) {
    Node *n = malloc(sizeof(Node));
    if (!n) { perror("malloc"); exit(1); }
    n->data = data;
    n->next = NULL;
    return n;
}

Node *push_front(Node *head, int data) {
    Node *n = make_node(data);
    n->next = head;
    return n;
}

Node *push_back(Node *head, int data) {
    Node *n = make_node(data);
    if (!head) return n;
    Node *cur = head;
    while (cur->next) cur = cur->next;
    cur->next = n;
    return head;
}

void print_list(Node *head) {
    for (Node *cur = head; cur; cur = cur->next)
        printf("%d ", cur->data);
    printf("\n");
}

Node *remove_val(Node *head, int val) {
    if (!head) return NULL;
    if (head->data == val) {
        Node *tmp = head;
        head = head->next;
        free(tmp);
        return head;
    }
    Node *cur = head;
    while (cur->next && cur->next->data != val)
        cur = cur->next;
    if (!cur->next) return head;
    Node *tmp = cur->next;
    cur->next = tmp->next;
    free(tmp);
    return head;
}

void free_list(Node *head) {
    Node *cur = head;
    while (cur) {
        Node *nx = cur->next;
        free(cur);
        cur = nx;
    }
}

int main(void) {
    Node *head = NULL;
    for (int i = 1; i <= 5; i++) head = push_back(head, i);  // 1 2 3 4 5
    head = push_front(head, 0);                                // 0 1 2 3 4 5
    print_list(head);                                         // 0 1 2 3 4 5
    head = remove_val(head, 3);                               // 0 1 2 4 5
    print_list(head);
    printf("len=%d\n", length(head));
    free_list(head);
    return 0;
}
```

> 把 `length` 补上即可跑（10.5）。开 `-fsanitize=address` 编译，能验证你没有泄漏/越界。

## 易错点

1. **忘返回新 head**：返回值函数里改了形参 head 不生效，调用方必须 `head = push_front(head, x)`。
2. 插入/删除找错节点：要操作「前驱」(i-1)，不是当前节点。
3. 接指针顺序反：先接新节点的 next，再改前驱的 next。
4. 删头没单独处理（头没前驱）。
5. free 后还用节点。
6. 空链表 / 单节点链表边界：永远先想「空表会怎样」「只有一个节点会怎样」。

## 例题

**例 10.1** 头插 1,2,3 后链表顺序？答：`3 2 1`（头插反序）。

**例 10.2** 尾插函数里 `while (cur->next != NULL)` 改成 `while (cur != NULL)` 会怎样？
答：`cur` 会走到 NULL，然后 `cur->next = n` 解引用 NULL，崩。尾插必须停在最后一个**节点**上（`cur->next != NULL`）。

**例 10.3** 画图：链表 `1→2→3`，在 2 后插 9。答：找前驱=节点1？不对，插在 2 后面，前驱是节点 2（`cur` 走到 2）。`n->next = 2->next(=3)`; `2->next = n`。结果 `1→2→9→3`。

**例 10.4** 删除函数里为什么删头要单独写？
答：因为删头要修改 `head` 本身，而其他节点只要改前驱的 next。头没有前驱，逻辑不同。

## 小练习

1. 补全上面完整代码里缺的 `length`。
2. 写 `Node *reverse(Node *head)`，原地反转单链表（头变尾、尾变头）。提示：用三个指针 prev/cur/next。
3. 写 `Node *insert_sorted(Node *head, int data)`，在**有序**链表中插入 data 保持有序。
4. 写 `Node *get_middle(Node *head)`，一次遍历找中间节点（快慢指针：快走两步慢走一步）。

<details><summary>答案</summary>

```c
// 1
int length(Node *head) {
    int n = 0;
    for (Node *c = head; c; c = c->next) n++;
    return n;
}

// 2 反转（经典，画图！）
Node *reverse(Node *head) {
    Node *prev = NULL, *cur = head;
    while (cur) {
        Node *next = cur->next;   // 暂存下一个
        cur->next = prev;         // 当前指向前一个（反转方向）
        prev = cur;                // prev 前进
        cur = next;                // cur 前进
    }
    return prev;                   // prev 是新头
}
// 过程：NULL<-1  2->3->NULL  逐个把箭头反过来

// 3 有序插入
Node *insert_sorted(Node *head, int data) {
    Node *n = make_node(data);
    if (!head || data < head->data) {   // 插头
        n->next = head;
        return n;
    }
    Node *cur = head;
    while (cur->next && cur->next->data < data)
        cur = cur->next;                // 找到第一个 >= data 的前驱
    n->next = cur->next;
    cur->next = n;
    return head;
}

// 4 快慢指针找中点
Node *get_middle(Node *head) {
    Node *slow = head, *fast = head;
    while (fast && fast->next) {
        slow = slow->next;
        fast = fast->next->next;
    }
    return slow;
}
```
