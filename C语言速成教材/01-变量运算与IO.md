# 01 变量、运算与输入输出

基础篇第一节，快讲。重点是 **类型转换** 和 **scanf/printf 格式**，这两块最易错。

## 1.1 变量与数据类型

变量 = 内存里一块有名字的盒子。**用之前必须先声明**。

```c
int age = 20;          // 整数
double pi = 3.14159;   // 双精度浮点（数据结构里用 double，别用 float）
char c = 'A';          // 单个字符
long long big = 9000000000LL;  // 更大的整数
```

408 最常用：`int`、`double`、`char`。偶尔 `long long`。

| 类型 | 字节 | 备注 |
|------|------|------|
| `char` | 1 | 字符也是小整数，`'A'` == 65 |
| `int` | 4 | 最大约 ±21 亿，超了会溢出（变负数） |
| `long long` | 8 | 大数，输出用 `%lld` |
| `double` | 8 | 浮点首选 |

### 常量

```c
const int MAX = 100;     // 不可修改
#define MAX 100         // 宏，编译时替换。两者数据结构代码都常见
```

## 1.2 运算符

**算术**：`+ - * / %`
```c
7 / 2      // 3   整数除整数=整数，截断！
7.0 / 2    // 3.5
7 % 2      // 1   取余，只对整数
```

**自增自减**：`i++`（先用后加）/ `++i`（先加后用）。单独一行写哪个都一样。

**关系**：`==  !=  <  >  <=  >=`。注意 `==`（判断相等）vs `=`（赋值）。

**逻辑**：`&&`（且） `||`（或） `!`（非）。
```c
if (a > 0 && b > 0) { ... }     // a、b 都 > 0
if (p != NULL && p->data > 0)  // 短路：p 为空时不访问 p->data，链表/树天天用
```

**复合赋值**：`x += 1` 等价 `x = x + 1`，同理 `*=  -=  /=  %=`。

## 1.3 类型转换（重点易错）

**隐式**：小类型自动提升为大类型。
```c
int i = 3;
double d = i;            // 3.0
double r = i / 2;        // 先算 3/2=1（整数除法），再 1→1.0 ❌ 不是 1.5
double s = i / 2.0;      // 3.0/2.0 = 1.5 ✅
```

**强制转换**：
```c
double avg = (double)total / n;   // 先把 total 转成 double 再除
```
**口诀：算平均分先转分子**，否则整数除法先截断。

## 1.4 printf 输出

```c
printf("a=%d, x=%.2f\n", a, x);
```
占位符：

| 占位符 | 含义 |
|--------|------|
| `%d` | int（d = **d**ecimal 十进制） |
| `%ld` / `%lld` | long / long long（**l** = long，加长版整数） |
| `%f` | double 输出（f = **f**loat，`%.2f` 两位小数） |
| `%c` | 单个字符（c = **c**haracter） |
| `%s` | 字符串（s = **s**tring） |
| `%p` | 指针地址（p = **p**ointer） |

> **记忆法**：大多是英文全称首字母——`d/c/s/p/f` 都是。注意 `d` 是 **decimal（十进制）**，不是 digit。它强调「用十进制打印」，因为整数还能用别的进制：
> `%d`（十进制）`%o`（八进制 octal）`%x`（十六进制 hex），这三个都是"按什么进制显示"。
> `ld`/`lld` 就是给 `d` 前面加 `l`（long），一个 `l` 是 `long`，两个是 `long long`。

## 1.5 scanf 输入

```c
int a;
scanf("%d", &a);   // 必须 &a！
double x;
scanf("%lf", &x);   // double 用 %lf 读
```

两个关键点：
- **必须加 `&`**（取地址），忘了程序崩溃。学完指针你懂为什么。
- **读 double 用 `%lf`，但输出 double 用 `%f`**。背下来。

读多个：`scanf("%d %d", &a, &b);`，输入 `3 5` 即可（空格/换行都行）。

`scanf` 返回成功读入的项数，可用来判断输入结束：
```c
while (scanf("%d", &a) == 1) { ... }   // 读到 EOF 停
```

读字符串不加 `&`（数组名就是地址），用 `%9s` 防溢出（留位给 `\0`）：
```c
char s[10];
scanf("%9s", s);
```

## 易错点

1. `7 / 2 == 3`，要小数写 `7.0 / 2`。
2. `=` 赋值 vs `==` 判断，`if (a = 5)` 永远为真。
3. scanf 读 double 用 `%lf`，printf 输出 double 用 `%f`。
4. scanf 漏 `&`。
5. 中文标点（全角分号括号）会报一堆错，代码标点必须英文半角。
6. `int` 算阶乘溢出，用 `long long` + `%lld`。

## 例题

**例 1.1** 输出？
```c
int a = 7, b = 2;
printf("%d %.2f %d\n", a/b, a/(double)b, a%b);
```
解析：`a/b=3`，`a/(double)b=3.5` → `3.50`，`a%b=1`。输出 `3 3.50 1`。

**例 1.2** 改错：求平均分
```c
int total = 90, n = 4;
double avg = total / n;
```
答：`total / n` 整数除法=22 再转 22.0。应 `double avg = (double)total / n;` → 22.5。

## 小练习

1. 读入秒数（int），输出「X分Y秒」（如 130 → `2分10秒`）。
2. 读入华氏 F（double），转摄氏 `C = 5.0/9.0*(F-32)`，保留 2 位。
3. 读入两个整数，输出较大者。

<details><summary>答案</summary>

```c
// 1
#include <stdio.h>
int main(void) {
    int s; scanf("%d", &s);
    printf("%d分%d秒\n", s/60, s%60);
    return 0;
}
// 2
#include <stdio.h>
int main(void) {
    double F; scanf("%lf", &F);
    printf("%.2f\n", 5.0/9.0*(F-32));
    return 0;
}
// 3
#include <stdio.h>
int main(void) {
    int a, b; scanf("%d %d", &a, &b);
    printf("%d\n", a > b ? a : b);
    return 0;
}
```
