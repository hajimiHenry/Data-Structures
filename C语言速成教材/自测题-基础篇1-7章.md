<details><summary>🔑 答案与解析（做完再看！）</summary>

**1.1** `3 3.50 1`。`a/b` 整数除法=3；`a/(double)b`=3.5→`3.50`；`a%b`=1。
**1.2** `total/n` 两个 int，整数除法=22 再转 22.0。改 `(double)total / n` → 22.5。
**2.1** `printf("%d\n", a > b ? a : b);`
**3.1** `void swap(int *a,int *b){int t=*a;*a=*b;*b=t;}` 调用 `swap(&x,&y)`。
**3.2** `0`。值传递，改的是副本。
**3.3** `void double_it(int*p){*p*=2;}` 改的是 p 指向的 int（数据），一级够。`void set_null(int**pp){*pp=NULL;}` 改的是指针本身，要升一级用二级。
**4.1** 双指针：`for(i=0,j=n-1;i<j;i++,j--){t=a[i];a[i]=a[j];a[j]=t;}`
**4.2** 标准 double for + 相邻交换。
**4.3** `sizeof(a)` 是指针大小（8 字节），不是数组总大小。函数内数组已退化成指针，拿不到长度，所以必须单独传 n。
**5.1** `1 3 2`。`*p`=a[0]=1，`*(p+2)`=a[2]=3，`p[1]`=a[1]=2。
**5.2** `20 20`。`*p=*q` 即 x=y。之后 `p=q` 让 p 也指 y，`*p`=20，但 x 没被碰（还是 20，因为上一步 x 已经被改成 20 了——注意顺序！）。
**5.3** `for(int*p=a;p<a+5;p++) sum+=*p;`
**5.4** `99`。`**pp`→`*p`→x，改的就是 x。
**5.5** `Node*node=malloc(sizeof(Node)); node->data=val; node->next=*head; *head=node;`
**6.1** `a.id`（真身用 `.`）；`p->id`（地址用 `->`）；`p[0].id`（p[0]=*(p+0) 是真身，用 `.`）。
**6.2** `t->s++; if(t->s==60){t->s=0;t->m++; if(t->m==60){t->m=0;t->h++; if(t->h==24)t->h=0;}}`
**7.1** `int n=0;while(s[n])n++;return n;`（遇 `\0` 即 0 停）。
**7.2** B 对。A 比的是两个数组的首地址，不同数组地址不同，几乎永不相等。比内容必须 strcmp。
**7.3** `int i=0,j=strlen(s)-1;while(i<j){char t=s[i];s[i]=s[j];s[j]=t;i++;j--;}`
**7.4** s 是野指针没指向有效内存，strcpy 往随机地址写会崩。修：`char s[20];` 或 `s=malloc(20);`。
**7.5** `strcpy(name,"Alice"); strcat(name,"!!");`
