# 02 控制流：判断与循环

程序的三种结构：**顺序、分支、循环**。这章快讲。

## 2.1 判断 if / else if / else

```c
if (score >= 90) {
    grade = 'A';
} else if (score >= 60) {
    grade = 'B';
} else {
    grade = 'C';
}
```
- 条件为「非 0」即真，`0` 为假。所以 `if (p)` 等价 `if (p != NULL)`。
- 一条语句可省 `{}`，但**永远写 `{}`**，防后面加语句出错。

## 2.2 switch

```c
switch (op) {
    case '+': r = a + b; break;
    case '-': r = a - b; break;
    default:  r = 0; break;   // 没匹配的走这里
}
```
- `case` 后是**常量**，不能是变量。
- **忘 `break` 会「穿透」**继续执行下一个 case，是经典 bug。除非你故意穿透。
- `default` 处理意外值。

## 2.3 三目运算符

```c
int max = (a > b) ? a : b;   // 条件 ? 真值 : 假值
```
简单的二选一用它比 if 短。

## 2.4 循环 for / while / do-while

```c
// for：知道次数
for (int i = 0; i < n; i++) { ... }

// while：先判后做，可能一次都不执行
while (条件) { ... }

// do-while：先做后判，至少执行一次
do { ... } while (条件);
```

读 n 个数求和：
```c
int n; scanf("%d", &n);
int sum = 0;
for (int i = 0; i < n; i++) {
    int x; scanf("%d", &x);
    sum += x;
}
```

读到 EOF 为止（不知道多少个）：
```c
int x;
while (scanf("%d", &x) == 1) {
    sum += x;
}
```

## 2.5 break / continue

- `break`：跳出**当前**循环。
- `continue`：跳过本次剩余，进下一轮。
- 嵌套循环里 `break` 只跳一层，不是全部。

```c
for (int i = 2; i < n; i++) {
    if (n % i == 0) { is_prime = 0; break; }   // 发现能整除就不是素数，跳出
}
```

## 易错点

1. `if (a = 5)` 赋值恒真，应为 `==`。
2. switch 忘 `break` 导致穿透。
3. `for (int i = 0; i <= n; i++)` 越界——多数情况是 `< n`，想清楚边界。
4. 死循环：`while (i < n) { ... }` 里忘了 `i++`。
5. `continue` 在 while 里跳过更新语句会导致死循环——把更新放 continue 之前，或用 for。

## 例题

**例 2.1** 判断素数。
```c
int is_prime(int n) {
    if (n < 2) return 0;
    for (int i = 2; (long long)i * i <= n; i++)   // 只判到 sqrt(n)，i*i 防溢出
        if (n % i == 0) return 0;
    return 1;
}
```
关键：判到 `√n` 即可；`i*i <= n` 用 `long long` 防大 n 时 `i*i` 溢出变负。

**例 2.2** 求 1 到 n 中能被 3 或 5 整除的数之和。
```c
int sum = 0;
for (int i = 1; i <= n; i++)
    if (i % 3 == 0 || i % 5 == 0) sum += i;
```

**例 2.3** 下面输出什么？
```c
int i;
for (i = 0; i < 5; i++) {
    if (i == 2) continue;
    if (i == 4) break;
    printf("%d ", i);
}
```
答：`0 1 3 `。i=2 跳过不打印；i=4 break 退出。

## 小练习

1. 读入 n，输出九九乘法表前 n 行。
2. 读入一个正整数，逆序输出各位数字（如 1230 → `0321`）。
3. 求 100 以内所有素数。

<details><summary>答案</summary>

```c
// 1
for (int i = 1; i <= n; i++) {
    for (int j = 1; j <= i; j++)
        printf("%d*%d=%-4d", j, i, i*j);   // %-4d 左对齐宽 4
    printf("\n");
}

// 2
int n; scanf("%d", &n);
if (n == 0) { printf("0\n"); return 0; }
while (n > 0) {
    printf("%d", n % 10);
    n /= 10;
}
printf("\n");

// 3
for (int n = 2; n <= 100; n++) {
    int prime = 1;
    for (int i = 2; (long long)i*i <= n; i++)
        if (n % i == 0) { prime = 0; break; }
    if (prime) printf("%d ", n);
}
```
